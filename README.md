# SymfonyNews

## Objective
The goal of this project to build a small web application to get to play around with some features of Symfony 5 and Twig.


## Technologies used
* Symfony
* Twig
* JavaScript
* Boostrap 
* MySQL

## Main focuses 
* Routing, wildcard routes
* Controllers
* Annotations
* Recipes, aliases
* URLs generation
* Repositories and Twig views
* Log-in Form
* Registration Form
* Form Type and validation
